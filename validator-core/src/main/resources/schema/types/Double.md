# Double

## Description

Représente une valeur numérique décimale encodée en virgule flotante (par ex : `42.44545`, `1e7`,...)

## Format

La valeur doit être encodée en utilisant des `.` (pas de `,`)

## Mapping

| Langage    | Type      |
| ---------- | --------- |
| Java       | `Double` |
| JSON       | `number`  |
| PostgreSQL | `real` |
