# Date

## Description

Représente un jour, un mois et une année.

## Formats

Les formats suivants sont supportés implicitement lors de la validation :

* `yyyyMMdd`
* `dd/MM/yyyy`
* `yyyy-MM-dd`
* `yyyy/MM/dd`

## Mapping

| Langage    | Type             |
| ---------- | ---------------- |
| Java       | `java.util.Date` |
| JSON       | `string`         |
| PostgreSQL | `date`           |
